# marugame-udonuts-site

Vercel で公開しやすい `Vite + React + Tailwind CSS` 構成のランディングページです。

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## GitHub と Vercel

1. GitHub で新規リポジトリを作成
2. このディレクトリの内容を push
3. Vercel で GitHub リポジトリを import
4. Framework Preset は `Vite` を選択
5. Build Command は `npm run build`、Output Directory は `dist`

## 備考

- 元コードをベースに、Vite でそのまま動くよう不足ファイルを補完しています。
- 絵文字と外部リンク属性など、描画やビルドに必要な軽微な修正を入れています。
